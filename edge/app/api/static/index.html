系统配置

舒适度与寻优约束

寻优可调：冷水出水温度（查表±微调）、冷冻泵/冷却泵频率与台数。固定不调：主机负荷、冷却塔频率/台数/功率（定额保持现场当前）。泵边界与「设备配置」频率范围取交集。

Excel 批量导入缺省值

Excel 缺少对应列时使用以上值；末端风机填 0 表示未知，模型将使用下方「缺省末端功率」。

能耗模型参数

包含舒适度、安全约束、Excel 缺省值、定时寻优与能耗模型参数。

寻优输入参数

支持现场多级表头趋势文件；将解析「运行状态=运行」的最后一行并填充。寻优只调冷水温度与冷冻/冷却泵；主机负荷与冷却塔定额不变。
